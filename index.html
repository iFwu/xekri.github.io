Hexo 的一小步，前端的一大步～

标题当然是对我来说的啦。其实想做一个个人博客很久了，一直没有付出实践，主要是懒……

域名用意

ZenBar.me 是这个博客对外的域名。Zen 就是禅，虽然有些人可能觉得禅很玄，但是我觉得它就在我们生活之中。其实我一开始想注册的域名是 ZenBar.io，它是我的一个计划，希望能向码猿和开发者们介绍『正念』这一在禅宗中十分精妙的观念。
不过由于 io 域名最少也要两百多块 RMB，我的建站水平也不足以应付诸如 Web 游戏这样的构想，所以这个计划暂时搁置。最近看到阿里云上 .me 域名十分便宜，我就买了一个，也正式开始了我的建站计划。

建站过程

其实通过一些打包好的程序，也许一个脚本文件就能免费的完成一整个博客的搭建过程。不过这样就没有意思啦。还是自己亲自动手，这样才能体会到一个网站的『前世今生』。
最大的障碍你们造是什么，是 Git 分布式版本管理系统的学习……其实原来一直就想学习，但是还是害怕命令行以及可能会出现的复杂的报错。不过好在现在教程资源丰富，很快我就上了贼船。其他的网上 Hexo + GitHub.io Page 的建站文章一搜全是的，我就不献丑啦。

建站计划

  • 写 MD 文章（正在写的这份）
  • 通过 SourceTree 推送到 GitHub 以及 BitBucket ，并测试哪个页面更快
  • 在 Coding.net 上镜像一份（或者通过七牛云储存进行镜像加速）
  • 写博客
    以上。
